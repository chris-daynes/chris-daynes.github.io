What is 'scope' and how it works in JavaScript.

Scope refers to what variables are able to be seen (and therefore there content accessed) by other parts of a program. Scope is described as either global (can be seen by all functions) or local (only seen by the function within which the variable is declared).

So all functions have access to variables of global scope, but only variables declared within functions are only visible to that particular function. They have local scope.

There is one other little quirk and that is that functions can see the varibles of their parent. Typically this means global scope but when a function is nested within another function, the nested function can see the varibale above it but it has no scope beyound it's parent function. Like a local global scope!
