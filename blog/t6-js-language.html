6.5 Technical Blog

Explain the following uses of these Javascript syntax characters.

  • Parentheses ()

    • Parentheses are used in two ways in Javascript. The first and most obvious is that they are used to group data in complex mathmatical equations. Just like you would have learnt at school. An example of this maybe (2 + 3) * 5.

    The second place one finds parentheses used is in functions. They are used here to both indicate that the thing is a function and to group any peices of information that needs to be passed into the function. iefunction(name, age).

  • Brackets []

    • Brackets are used in Javascript to group data (like parentheses) but the use is with arrays and in indexing or localising data in objects. The [] is used to group array data like var array = ["thing1", thing2];. With arrays it encloses a numerical value to show the exact postition of that array element ie array[2]. With objects, it again is used for locatising data but in the object it's use is not numberical but contains property strings ie object["name"].

  • Braces {}

    • Braces are used to enclose a code block.

    var func = function() { do something };

    They are also used show that a particular group of data is an object

  • Single quotes ''
  • Double quotes ""

    • Single and Double quotes are used interchangably. They indicate that a peice of data is a string. ie var string = "This is a string"
